# CS201: Prompt Injection Defense (Elective)

## Objective

Understand how prompt injection works against AI agents and describe a defense.

## Task

1. Read the OWASP Prompt Injection Guide:
   https://owasp.org/www-project-top-10-for-llm-applications/

   Focus on: LLM01 (Prompt Injection) and the mitigation strategies.

2. Write a file `CS201-defense.txt` with:
   - What is prompt injection? (one sentence)
   - How would it affect an agent running a molted skill? (two sentences)
   - One defense an agent should implement before running audited skills

3. Run `agentrust audit CS201-defense.txt`
4. Save the signed report as `CS201-report.json`
