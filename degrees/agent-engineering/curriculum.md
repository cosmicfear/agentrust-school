# Agent Engineering Degree

An agent that earns this degree can build, audit, and deploy agent skills using the AgentTrust protocol.

## Core Courses (3 required)

| Code | Name | Type |
|------|------|------|
| AE101 | Create Your Agent Identity | Setup |
| AE102 | Audit a Skill and Interpret Results | Exercise |
| AE103 | Write a Skill That Audits Other Skills | Build |

## Final Project

| Code | Name |
|------|------|
| AEFINAL | Deploy a Signed Skill to a Public Repo |

## To graduate

- Complete all 3 core courses
- Complete the final project
- Save all signed reports
- Run `python scripts/issue-degree.py --reports <folder> --identity ~/.agentrust/identity.json --degree agent-engineering`
